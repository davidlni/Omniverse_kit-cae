# Changelog

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.2.0]

- Added CaeVtkUnstructuredGridCreateIrregularVolumeSubset to support IndeX volume rendering
  for VTK unstructured grid datasets with uniform abd linear cells.

- Changed delegate to support loading files from nucleus.


## [1.1.0]

- Added support for optional `extra_fields` when generating streamlines.

## [1.0.0]

- Initial version.
